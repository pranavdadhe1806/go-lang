# Go-Lang Learning Journey 🚀

Welcome to my Go-Lang repository! This repository serves as a personal documentation of my journey learning the Go programming language from scratch. 

My goal is to progress from the fundamentals all the way to advanced concepts, building projects and exploring the language's capabilities along the way.

## 🎯 About This Repository

This is an ongoing process. As I learn and experiment, I will be adding code snippets, exercises, notes, and projects to this repository. It acts as both a learning journal and a portfolio of my Go development skills. All topics are organized in sequential order to track the learning path.

## 📚 What's Inside

- **Basics**: Variables, data types, control structures, and functions.
- **Intermediate**: Arrays, slices, maps, structs, pointers, methods, interfaces, and error handling.
- **Advanced**: Concurrency (goroutines and channels), testing, context, generics, and other advanced topics.
- **Projects**: A dedicated space for real-world applications built to solidify my understanding.

## 🚀 Getting Started

To run any of the Go files in this repository, you'll need to have Go installed on your machine. You can download it from [golang.org](https://golang.org/).

Navigate into any topic folder and use the `go run` command. For example:

```bash
cd 01-hello-world
go run hello.go
```
